1.เตรียม Repository
	1.1 สร้าง Repository ของทีมบน GitHub Classroom และกำหนดชื่อกลุ่ม
	1.2 เพิ่มสมาชิกในทีมเป็น Collaborators 

2.เริ่มต้นสร้างโปรเจกต์
	2.1 เพิ่มคำอธิบายเกี่ยวกับโปรเจกต์ของทีมลงใน Readme.md ใน Repository ของทีม
	2.2 ทำการโคลน Repository ของทีมลงใน Workspace ของตนเอง
	2.3 สร้างโฟลเดอร์ ชื่อ AboutMyTeam และ Project 
	2.4 สร้างไฟล์ TeamWorkflow.md ใน AboutMyTeam 
	2.5 เพิ่มรายละเอียดของสมาชิกในทีมและอธิบาย Workflow ในการทำงานของทีม ลงใน TeamWorkflow.md
	2.6 Commit งานและใส่คำอธิบาย

3.สร้าง Branch เพื่อแยกงานของแต่ละคน
	3.1 สมาชิกแต่ละคนสร้าง branch ของตัวเอง โดยใช้ชื่อ branch ตามชื่อจริงและรหัสนักศึกษา 4 ตัวท้าย
	3.2 สมาชิกทุกคนทำงานเฉพาะใน branch ของตัวเอง เพื่อลดความเสี่ยงในการทับไฟล์กัน
	
4. เพิ่มไฟล์ข้อมูลส่วนตัว
	4.1 สมาชิกแต่ละคนสร้างไฟล์ด้วยชื่อจริงตามด้วยรหัส 4 ตัวท้าย ตามด้วย.md
	4.2 เพิ่มข้อมูลของตนเอง
